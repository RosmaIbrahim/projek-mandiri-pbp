1. Implementasi BGP (Border Gateway Protocol)

Diagram Topologi Lab BGP

BGP, singkatan dari Border Gateway Protocol, adalah protokol perutean untuk internet yang menentukan jalur terbaik bagi data untuk melakukan perjalanan di antara jaringan-jaringan yang berbeda (Sistem Otonom). BGP bekerja seperti sistem GPS untuk internet, yang memungkinkan router untuk bertukar informasi tentang rute yang tersedia sehingga paket data dapat dikirim secara efisien dari satu jaringan ke jaringan lain, bahkan melintasi berbagai belahan dunia.

Tantangan & Solusi

  • Tantangan: Ketidakstabilan Rute (Flapping), rute yang terus-menerus diumumkan dan ditarik kembali dapat membebani CPU router.
  • Solusi: BGP Monitoring, pengawasan real-time untuk mendeteksi anomali atau flapping rute secepat mungkin.

Tools yang Digunakan

Cisco IOS, GNS3/EVE-NG, Wireshark (untuk verifikasi).

2. Simulasi Alarm Kebakaran Otomatis dengan Arduino dan Sensor Asap/Api

Rancangan pada Arduino

Proyek implementasi sistem sederhana menggunakan Arduino Uno untuk mendeteksi adanya asap (menggunakan sensor MQ-2) dan/atau nyala api (sensor Flame). Ketika deteksi terjadi, sistem akan mengaktifkan alarm (buzzer) dan memberikan notifikasi visual melalui LED.

Implementasi Utama

  • Pemrograman sketch Arduino untuk membaca input dari sensor digital dan analog.
  • Konfigurasi threshold kepekaan sensor MQ-2 untuk meminimalisir false alarm.
  • Melakukan wiring (pengawatan) yang benar antara sensor, LED, Buzzer, dan Board Arduino.
  • Pengujian sistem alarm dan responsivitas sensor terhadap sumber api/asap simulasi.

Tools & Komponen yang Digunakan

Arduino IDE, Arduino Uno, Sensor Asap (MQ-2), Sensor Api (Flame Sensor), Buzzer Aktif, LED, Resistor.

3. Konfigurasi Firewall Sederhana

Diagram implementasi Firewall Stateful

Firewall Stateful pada perangkat Mikrotik untuk mengamankan jaringan internal. Fokus pada pemblokiran trafik masuk (input chain), pengamanan router itu sendiri, dan penerapan filtering berdasarkan protokol dan port number.

Implementasi Utama

  • Konfigurasi Firewall Filter Rules untuk memblokir akses internet ke IP tertentu (Forward Chain).
  • Penerapan rule anti-spoofing dan rule untuk mencegah serangan Brute Force ke layanan router (SSH/Winbox).
  • Mengatur NAT Masquerade dan Port Forwarding yang aman dan terseleksi.

Tools yang Digunakan

MikroTik RouterOS, Winbox, Cisco Packet Tracer (Simulasi Topologi), Nmap (Pengujian).
